The Code for Taco Cat

The Code is structured in three functions.

Get Values

Get Values stores the string entered by the user. The user string has any capitalization and special characters removed. It then calls the tacoCat and isPalindrome methods.

The screen is cleared, if a previous answer is visible.

Taco Cat

Taco Cat initializes an empty array to store the reversed string.

The index values of the string array are used as the boundaries for a decrementing for loop.

The loop moves backwards through the string and outputs each value into a new index value on the revString array. The revString array is returned.

Is Palindrome

Is Palindrome compares the original string to the reversed string. It targets the paragraph with the id "msg". It adds the revString array as inner html.

The alert box, with id "alert" removes the invisible class and adds the show class to display the reversed string inside the alert, along with a message "The string is a palindrome", or "The string is NOT a palindrome". If the string is a palindrome, the alert message is green, and if not, the alert is red.
